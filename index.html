Installation

VIA WordPress

  1. After your download log into your WordPress admin interface
  2. After login click on  Plugins from the left  hand menu
  3. Click on “Add new”
  4. Click on “Upload”
  5. Browse to the directory you downloaded the plugin to and click “Install Now”
  6. After Wordpress has finished unpacking the file click on “Activate Plugin”
  7. After the plugin has been activated you will notice a new menu item on the left hand navigation labelled NEX-Forms
     

VIA FTP

  1. After your download unzip Nex-Forms from your download .zip
  2. Open your FTP client
  3. Browse to /your_wp_instalation/wp-content/plugins/ server
  4. Uplaod Nex-Forms into this directory
  5. Go to your wordpress admin panel
  6. Go to plugins and activate NEX-Forms
  7. After the plugin has been activated you will notice a new menu item on the left hand navigation labelled NEX-Forms
     

Updating

  1. Download the plugin from CodeCanyon (always update the plugin from CodeCanoyn!)
  2. Delete the current version
  3. Repeat normal installation
IMPORTANT: You will not loose any data when updaing!

FIY: While on your CodeCanyon downloads page, check the box to be notified of updates.

Online Documantaion

  1. After instalation you will find a help button on the top stip of your Nex-Forms admin page. Click on it to view the latest docs.
  2. Alternatively, the same docs can be found here: http://basixonline.net/nex-forms/nex-forms-documentation/
NOTE: The docs are constantly updated and you always see the latest from your Nex-Forms admin page.
